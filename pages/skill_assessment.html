Current Skills Assessment & Gap Analysis

Evaluating your cybersecurity foundation and identifying areas for development

Current Skills (Based on Information Technologies with Cybersecurity Major)

As a graduate of Information Technologies majoring in cybersecurity from Deakin University, you likely have the following foundational skills:

General Cybersecurity Knowledge

  • Basic security principles and concepts
  • Understanding of common security threats and vulnerabilities
  • Knowledge of security controls and countermeasures
  • Familiarity with security frameworks and standards

Network Security

  • Understanding of network protocols and architectures
  • Basic knowledge of firewalls, IDS/IPS, and network monitoring
  • Experience with network traffic analysis

Security Assessment

  • Introduction to vulnerability assessment
  • Basic penetration testing concepts
  • Understanding of security auditing principles

Cryptography

  • Understanding of encryption algorithms and protocols
  • Knowledge of cryptographic controls
  • Basic implementation of cryptographic solutions

Security Governance

  • Introduction to security policies and procedures
  • Understanding of compliance requirements
  • Basic knowledge of risk management concepts

Skill Gaps for Security Analyst Position

Based on the job requirements, the following skill gaps need to be addressed:

Risk Assessment Expertise

  • Advanced risk assessment methodologies specific to government and enterprise environments
  • Experience with formal risk frameworks (NIST, ISO 27005, etc.)
  • Practical application of risk mitigation strategies

Vulnerability Scanning

  • Proficiency with enterprise vulnerability scanning tools
  • Experience analyzing scan results in large-scale environments
  • Ability to prioritize vulnerabilities based on business impact

API Development Security

  • Specific knowledge of API security best practices
  • Understanding of OWASP API Security Top 10
  • Experience with API gateway security configurations
  • Secure API design principles

AWS Security Tools

  • Hands-on experience with AWS security services (GuardDuty, Security Hub, IAM, etc.)
  • Cloud security architecture knowledge
  • AWS security compliance frameworks
  • Automated security scanning in AWS environments

Certifications

  • Industry-recognized cloud security certifications
  • AWS-specific certifications
  • Security analysis certifications

Enterprise Experience

  • Experience with large-scale data projects
  • Familiarity with agile methodologies in security contexts
  • Experience working in government security environments
